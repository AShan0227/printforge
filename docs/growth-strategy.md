## 核心卖点（用户在乎的）
1. **"100% watertight"** — 竞品最大痛点，Meshy 也只有 97%
2. **"Open source + local"** — 隐私敏感用户的刚需
3. **"Free"** — 开源免费，竞品 $20/月
4. **"One command"** — `pip install printforge` 极致简洁
5. **"Tested with Bambu Lab A1"** — 拓竹用户信任背书

## 用户发现的关键洞察
- Reddit 用户主要还在用 SVG→CAD→STL 的老方法
- AI image-to-3D 对大部分打印用户来说还是新鲜事
- "生成模型需要手动修复" 是最大抱怨 → 我们的 watertight 100% 直击痛点
- MakerWorld 自带 Image to 3D 功能但限制多（10次/月，质量一般）
